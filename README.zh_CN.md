### 快速开始

1. 安装依赖 ` npm i`

2. 配置mapbox token。 需要在根目录创建一个名为accessToken.js的文件

文件内容：

```
const accessToken = '你的 mapbox ACCESS_TOKEN';
export default accessToken;
```

3. `npm run dev` 即可运行本地demo
